# airborne-eo-tutorials
The primary GitHub repository for airborne SMCE Earth observation tutorials. Includes tutorials such as how to access data, process data, generate scientific summaries, and harmonize datasets.

Additional tutorials to explore data access and analysis can be found in the [618-hyrsense](https://github.com/GSFC-618/618-hyrsense) fork setup to run the NASA/ESIIL HYR-SENSE tutorials on the Airborne SMCE
